Automatic IIUM Schedule Formatter


Assalamualaikum semua! Apa khabar! Adakah kamu seorang pelajar CFS IIUM? Tak? Habis, kamu buat apa kat sini? Pergi kat 9gag. Jika kamu seorang pelajar IIUM, kamu pasti tau yang IIUM menggunakan CRS(Course Registration Slip) untuk pendaftaran kelas. Dan dalam slip tu tertulislah segala kelas anda beserta tempat dan waktunya.
Tapi, CRS diformat dalam format yang menyusahkan anda untuk periksa 'esok ada kelas apa?'. Yaknie, dia tak susun mengikut hari. So, berdasarkan kelecehan itu dan kemalasan saya, maka muncullah Automatic IIUM Schedule Formatter. So, basically, apa yang website ini buat ialah ia tukar CRS macam kat atas nie, kepada jadual macam kat bawah nie.
Of course, rekaan jadual tidak terhad kepada yang macam di atas sahaja. Kamu boleh customize jadual kamu sesuka hati kamu.
Automatic IIUM Schedule Formatter juga mempunyai theme gallery yang membolehkan kamu untuk mengubah gaya jadual kamu tanpa banyak pengetahuan tentang CSS, Javascript atau EJS template.
Tapi buat masa nie, tak ada banyak theme, so kalau kamu design yang bagus, tolong-tolonglah submit design anda

Cara menggunakan AISF

Untuk browser berasaskan webkit (Chrome atau Safari):
  1. Copy SETIAP SATU HURUF code kat bawah nie:
    javascript:(function(){var e=document.createElement('script');e.src = 'https://iiumschedule.appspot.com/static/scheduleformatter.js';e.type='text/javascript';e.addEventListener('load',function(){startscheduler()} );document.getElementsByTagName('head')[0].appendChild(e);})();
  2. Bukak korang punye Course Registration Slip (CRS) kat myiium.
  3. Paste code tadi kat url bar dan PASTIKAN IA DIMULAKAN DENGAN 'javascript:'
  4. Tunggu sat... nanti kat bawah korang akan jumpa. "Saving your schedule...". Dan kemudian dia kata "Done! Please click this link.". Korang pon click la link tersebut.

Untuk Firefox:
  1. Right click link Automatic IIUM Schedule Formatter.
  2. Click "bookmark this link".
  3. Bukak korang punye Course Registration Slip(CRS) kat myiium.
  4. Click firefox punye bookmark menu dan click bookmark yang korang baru buat tadi.
  5. Tunggu sat... nanti kat bawah korang akan jumpa. "Saving your schedule...". Dan kemudian dia kata "Done! Please click this link.". Korang pon click la link tersebut.

Disclamier

  • Mula-mula sekali, saya nak cakap yang pembikinan website ini tidak ada kena mengena dengan pihak IIUM. Use at your own risk. Walaupun bukannya ada risk apa pon.
  • Automatic IIUM Schedule Formatter hanya telah diuji terhadap CRS pelajar CFS(Centre for Foundation Studies) IIUM. Oleh itu, saya tak tau sama ada ia akan berfungsi untuk pelajar main campus IIUM atau tidak.
  • Sesiapa yang terer komputer akan perasan yang saya dengan teknik ini, anda akan melakukan javascript injection terhadap page IIUM. Kenapa perlu begitu? Asalnya saya ingat nak save aje, pastu parse, tapi rupanya tak boleh. Nampaknya IIUM Portal ada sistem security yang kuat sikit, so kena ada sikit hacking.
  • Bagi yang tak faham bahasa komputer, senang cakap, kalau ada sapa-sapa suruh korang buat macam nie, dan sumber code itu meragukan, macam "Facebook Unlike button", atau "Win an Ipad 2!", JANGAN BUAT APA YANG DISURUH. Basically, ia menyuruh anda untuk menjalankan javascript dari sumber yang tak dikenali dan kemungkinan javascript tersebut mengandungi virus, dan akhirnya muncullah virus di Facebook.
  • Bagi sesiapa yang terer komputer dan berminat untuk mengetahui bagaimana program ini berfungsi, silalah lakukan 'reverse engineer' terhadap code tersebut yang mana tidak di 'minified' pun. Kecuali untuk Styler.js yang mana juga digunakan di project saya yang lain iaitu YourAgregator
  • Dan by the way, disebabkan pelbagai cross-domain scripting restriction, saya telah mengambil jalan mudah dengan menyimpan jadual anda di server terlebih dahulu. So, ya saya boleh tahu jadual anda kalau saya tahu nama anda dan saya berminat untuk mengetahuinya. Tapi jangan risau, server akan automatically delete jadual anda selewat-lewatnya dalam 48 jam. So, lepas korang buat jadual, korang cuma ada 24 jam untuk save jadual anda. Lepas 24jam, ada kemungkinan link yang diberi untuk jadual anda tidak boleh digunakan lagi.
  • Itu tak termasuk theme, lah. Theme akan kekal, sehingga mana saya nak delete theme tersebut.
